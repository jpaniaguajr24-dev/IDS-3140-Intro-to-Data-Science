# Final Project - Data Science Classification Pipeline

## Overview
This final project applied the full data science process, including: data cleaning, exploratory analysis, feature preparation, machine learning models, and performance evaluation.

The goal was to build predictive models and compare results to determine the best solution.

## Objectives
- Import and clean raw data
- Explore patterns and trends
- Prepare features for modeling
- Train multiple machine learning models
- Compare model performance
- Present final insights and conclusions

## Files Included

- `FinalProject.ipynb` → Jupyter Notebook containing full project workflow

## Tools Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- SQLite 

## Machine Learning Models

- k-Nearest Neighbors (k-NN)
- Decision Tree
- Random Forest
- Additional models as tested

## Skills Demonstrated

- Data cleaning
- Exploratory Data Analysis (EDA)
- Feature engineering
- Model training
- Model comparison
- Confusion Matrix analysis
- Accuracy, Precision, Recall
- Business insight reporting

## Evaluation Methods

- Train/Test Split
- Cross Validation
- Accuracy Score
- Precision / Recall / F1 Score
- Confusion Matrix

## Result

Multiple models were tested and compared. The strongest model was selected based on performance metrics and practical usefulness.

## Conclusion

This project demonstrates the complete end-to-end data science workflow from raw data to final prediction results.
