
# Assignment 1 - Data Cleaning with Python

## Overview
This assignment focused on cleaning and preparing messy data using Python and Pandas.

The dataset contained missing values, inconsistent formatting, and errors that needed to be fixed before analysis.

## Objectives
- Load CSV data into Python
- Inspect dataset structure
- Clean missing values
- Standardize text and numeric columns
- Correct formatting issues
- Export cleaned dataset

## Files Included

- `Assignment1.ipynb` → Jupyter Notebook with code and steps
- `dirty_cafe_sales.csv` → Original raw dataset
- `cleaned_cafe_sales.csv` → Final cleaned dataset

## Tools Used

- NumPy

## Skills Demonstrated
